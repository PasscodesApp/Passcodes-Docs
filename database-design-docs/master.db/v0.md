## Setup SQL

```sql
BEGIN TRANSACTION;

CREATE TABLE IF NOT EXISTS `passwords` (

    `password` VARCHAR(60) NOT NULL,
    `notes` VARCHAR(100),
    `createdat` DATE DEFAULT CURRENT_TIMESTAMP,

COMMIT;
```
