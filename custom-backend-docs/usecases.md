## Custom backend in Homelabs

You haveing home labs in your home means, you follow all of this guys on YouTube. 
- [Techno Tim](https://youtu.be/gPGf4Y8nQqM?si=WVfreMSdzwHriK8i), 
- [Network Chuck](https://youtube.com/@networkchuck),
- [Tony Teaches Tech](https://youtu.be/ZKfnGqMrnug?si=0ShEONnOYjyp_twS),
- [Christian Lempa](https://youtu.be/n-fAf2mte6M?si=351b2pFbpvCATiI-)

"Homelabs" here means, you have minimalist network, in your home. where you connect a group of computer connect 
and You probably also have a raspberry Pi 5. 

So this is great initial setup, now you use that Raspberry PI 5 or any other spare computer to host passcodes custom backend server.

We recommend you, buy a raspberry PI 5, but you do job with any computer that you can supply power to 24/7. and a machine that can stay up nights in and days out. 

We recommend raspberry PI 5, because it give a scalable architecture, and you then use shared custom backend passcodes server in your home. Also setup a NAS to save password in a more reliable place and try backing it up either manually or using a automated workflow.

Congratulations, now you can use the passcodes app that had shared passcodes backend for your own family and you also get full control of passcodes.
