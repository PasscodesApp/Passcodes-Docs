# Security Guide

Here, In this file, You can find the information about security concern that arise with saving your auth information in passcodes app.

!!! warning
    
    Firstly, The passcodes is open source project and is currently under active development. 
    So, one is soley responsible for share there information with passcodes application.

We as developers, recommend you to wait, for its stable release. before full trusting passcodes app with your passwords data.

## Consideration

While, saving your password in passcodes application. you should consider or should have knowledge of following things.

-   Project is license under MIT License. which means that application is given `as it is,  without any warranty or guarantee` and is openly/freely available to redistribution or modification. (but you need to have a copy of original license in it)
-   With passcodes, your password are as safe as you write it down on pieces of paper or in a text file digitally. which means that, `your information is not specializily protected`.

## Recommendation

Here, are the thing we recommend you to improve safety of your passwords.

-   lock the passcodes app, using your phone's settings app. it will make it harded for an attackers to see your passwords.

-   if you lock the app, then use a different password, than the one you have on your lock screen. it make it even more safer.

-   also, if you don't use the app often, it's recommended, to hide the app using your phone's settings. (it is bit inconvenient but at the end of day it pays off as safety)
